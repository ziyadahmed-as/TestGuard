About Import Templates

User import templates help you standardize the process of bulk user creation. Each template defines:

  • Required fields that must be present in the CSV file
  • Optional fields that can be included
  • Field descriptions and formatting guidelines
  • A downloadable template CSV file for easy data entry

To use a template, download it, fill in the user data, and then use the bulk upload feature.
